# 第三组-程序设计实践项目

##  项目简介：基于python的坦克大战

## 分工

1. cwq：游戏核心
职责范围：
管理 MainGame 类和整体游戏协调
实现主游戏循环 (startGame 方法)
处理事件系统 (getEvent 方法)
协调各组件间的交互
管理游戏状态和流程控制
2. zhx：坦克系统
职责范围：
基础 Tank 类实现
MyTank (玩家坦克) 功能开发
EnemyTank AI行为逻辑
坦克移动和碰撞检测机制
坦克方向控制和显示功能
3. wyc：武器战斗系统
职责范围：
Bullet 类的完整实现
射击机制（玩家和敌人）
子弹轨迹和移动逻辑
子弹与坦克、墙体的碰撞检测
弹药管理和限制系统
4. wsq：环境与特效
职责范围：
Wall 类实现和障碍物系统
地图环境和阻挡机制
Explode 类和视觉特效系统
爆炸动画和渲染效果
环境元素的生命周期管理
5. xhb：音频界面
职责范围：
Music 类和音频系统
用户界面元素设计
文字渲染和显示功能 (getTextSurface)
游戏菜单和覆盖层
音效管理和播放控制
